DIAGNÓSTICO: TELA BRANCA
========================

CAUSAS POSSÍVEIS:
-----------------
1. Erro de sintaxe no JavaScript
2. Biblioteca não carregou (React, XLSX, html2pdf)
3. Cache do navegador
4. Arquivo corrompido durante edição

SOLUÇÃO RÁPIDA:
---------------

OPÇÃO 1: Verificar erros no Console
   1. Pressione F12
   2. Vá na aba "Console"
   3. Veja erros em vermelho
   4. Me envie screenshot ou copie o erro

OPÇÃO 2: Limpar tudo e recarregar
   1. Feche o navegador completamente
   2. Pare o servidor (Ctrl+C)
   3. Delete o cache do navegador:
      - Chrome: Ctrl+Shift+Delete
      - Marque "Imagens e arquivos em cache"
      - Clique em "Limpar dados"
   4. Reinicie o servidor
   5. Abra http://localhost:8000

OPÇÃO 3: Usar backup
   1. Há um arquivo app-backup.js na pasta
   2. Se quiser restaurar, renomeie:
      - app.js → app-novo.js
      - app-backup.js → app.js
   3. Reinicie servidor

OPÇÃO 4: Verificar se scripts carregaram
   No Console (F12), digite:
   
   React
   
   Se aparecer "undefined", React não carregou
   
   XLSX
   
   Se aparecer "undefined", XLSX não carregou

PRÓXIMO PASSO:
--------------
Me envie o que aparece no Console (F12)
para eu saber exatamente qual é o erro!
