# Diagnóstico Financeiro

Ferramenta de análise financeira para pequenas e médias empresas.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Preencha o nome da empresa e mês de referência
3. Faça upload do extrato bancário (Excel ou CSV)
4. Classifique as movimentações
5. Preencha os dados do Pulmão Financeiro
6. Preencha os Custos Financeiros (opcional)
7. Gere o relatório em PDF

## Funcionalidades

- **Pulmão Financeiro**: Calcula quantos dias a empresa sobrevive com o caixa atual
- **Anomalias A1, A2, A3**: Detecta problemas de fluxo de caixa
- **T1 - Taxa Real de Vendas**: Analisa custos de cartão/gateway
- **T2 - Antecipação Mascarada**: Identifica custos ocultos de antecipação
- **T3 - Juros Pulverizados**: Soma custos bancários ocultos
- **Tema Claro/Escuro**: Alterne clicando no ícone sol/lua
